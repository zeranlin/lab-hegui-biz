# 质量检查提示词

审查完成前必须自检：

1. 文件画像是否完整。
2. 预算金额是否与原文一致。
3. 采购方式和评标方法是否与原文一致。
4. 采购品目和所属行业是否核对。
5. 是否逐章审查。
6. 风险点正文编号是否与元数据数量一致。
7. 每个风险点是否都有原文内容。
8. 原文内容是否只保留招标文件原文。
9. 每个风险点是否有关联审查点。
10. 缺失类风险是否说明应有内容和未见内容。
11. 审查报告是否记录运行记录路径。
12. 运行记录是否记录审查报告路径。
13. 审查产物是否都写入业务输出目录。
14. 是否确认未修改 LLM Wiki。
15. 是否确认未对 LLM Wiki 运行 `./lint`、`./ingest`、`./query` 等维护命令。
16. 是否读取并执行外部执行主体招标文件审查指引。
17. 是否生成 `PROP-A01` 至 `PROP-A23` 动作清单。
18. 每个物业动作是否都有 `命中`、`未命中` 或 `待确认` 状态。
19. 每条风险是否有关联动作ID。
20. 是否按风险原子化规则拆分，不把社保、证明材料、合同空白等合并进大风险。
21. 是否执行质量门规则并记录二次反查结果。
22. 是否确认未读取同一项目既有审查记录、运行记录、人工复盘记录或知识更新申请作为答案来源。

如任何检查不通过，应先修正再交付。
