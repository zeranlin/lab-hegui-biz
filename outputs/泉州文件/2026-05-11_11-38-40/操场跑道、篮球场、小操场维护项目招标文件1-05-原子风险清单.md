# 05-原子风险清单

## 风险清单执行摘要

基于 `04-动作执行记录` 的审查结果，本项目（操场跑道、篮球场、小操场维护项目）存在多项风险。

**原子化执行说明**：
1.  **GEN-A02** 中“行业标注错误”与“中小企业政策适用”虽相关，但前者是事实认定错误，后者是政策执行风险，依据 `[[wiki/70-审查协议/风险原子化规则]]` 第4.7条（缺失类风险）及第2条（独立原文证据），拆分为独立风险。
2.  **GEN-A04** 中“全章实质性”与“异常低价裁量”属于不同维度的符合性审查风险，拆分为独立风险。
3.  **GEN-A05** 中“技术参数指向”包含“具体参数指向”和“指定品牌”两个独立信号，且指向性参数与品牌指定对竞争的限制机制不同，拆分为独立风险。
4.  **GEN-A06** 中“合同验收标准空白”与“引用标准过时/不规范”拆分为独立风险，前者为缺失类，后者为需求不明确。
5.  **GEN-A07** 中“合同模板错配”与“专用条款空白”拆分为独立风险，前者为属性错误，后者为关键信息缺失。
6.  **GEN-A10** 中“检测报告要求不一致”、“截图证明效力”、“SGS报告效力”、“原件复核后果”拆分为独立风险，分别对应证明材料要求不清、过度、不当及处罚过重。
7.  **GEN-A12** 中“专门预留与价格扣除冲突”拆分为独立风险。

## 原子风险明细

### 1. 采购标的所属行业标注错误，影响中小企业政策适用

- **结论类型**: 事实认定错误/政策适用风险
- **风险等级**: 高
- **原文位置**: 附2：采购标的一览表 第0090行
- **原文内容**: 所属行业 工业
- **问题说明**: 本项目采购内容为“操场跑道、篮球场、小操场维护项目”，涉及拆除、面层改造、排水、灯杆安装等，实质为工程施工或体育设施安装服务。根据《政府采购促进中小企业发展管理办法》，工程类项目对应行业应为“建筑业”，服务类项目对应行业应为“其他未列明行业”或具体服务行业。将运动场地维护标注为“工业”属于行业认定错误。本项目专门面向中小企业采购，行业标注错误将导致投标人依据“工业”标准出具的《中小企业声明函》无效，进而导致投标无效或政策适用错误，严重排斥潜在供应商。
- **来源动作ID**: GEN-A02
- **关联审查点**: [[wiki/25-风险审查点/采购标的所属行业错误或缺失]]；[[wiki/25-风险审查点/中小企业政策适用错误]]
- **审查依据**: 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）；《政府采购品目分类目录》
- **修改建议**: 修正采购标的一览表中的所属行业，根据项目实质内容（工程施工或体育设施安装服务）将其调整为“建筑业”或“其他未列明行业”，并确保与中小企业声明函填写口径一致。
- **是否需要人工复核**: 否

### 2. 符合性审查将非核心条款扩大为投标无效

- **结论类型**: 投标无效条款扩大化
- **风险等级**: 中高
- **原文位置**: 第四章 资格审查与评标 第0585行、第0590行
- **原文内容**: 
  - 0585: 《招标文件》“第五章 招标内容及要求 二、技术和服务要求”中的内容皆为实质性要求，投标文件必须满足，否则按无效投标处理。
  - 0590: 《招标文件》“第五章 招标内容及要求 三、商务要求”中的内容皆为实质性要求，投标文件必须满足，否则按无效投标处理。
- **问题说明**: 招标文件规定第五章“技术和服务要求”及“商务要求”中的**所有内容**皆为实质性要求，不满足即按无效投标处理。这种“全章实质性”的表述过于宽泛，未区分核心实质性条款（如★号条款）与一般性技术描述、售后服务承诺细节或非关键商务条款。这可能导致供应商因非核心条款的细微偏差（如一般性描述不符）而被直接否决投标，违反了87号令关于实质性要求应明确界定、不得随意扩大无效范围的原则，增加了投标无效风险，影响公平竞争。
- **来源动作ID**: GEN-A04
- **关联审查点**: [[wiki/25-风险审查点/符合性审查和投标无效条款扩大化]]；[[wiki/25-风险审查点/实质性条款设置过多或不清]]
- **审查依据**: 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十条、第六十条
- **修改建议**: 删除“第五章...中的内容皆为实质性要求”的概括性表述。明确列出哪些具体条款为实质性要求（通常以★号标识），其余非实质性条款不满足的，应按扣分或一般性偏离处理，不得直接判定为投标无效。
- **是否需要人工复核**: 否

### 3. 技术参数具有强指向性，排斥潜在供应商

- **结论类型**: 指向性参数
- **风险等级**: 高
- **原文位置**: 第五章 招标内容及要求 第1243-1266行
- **原文内容**: 
  - 1243-1266: 5.人造草坪及减震垫技术参数要求... 纤维组成 双开槽形直曲混织草 ... 纤维丹尼数 17000Dtex ... 纤维草高度 30±2mm ... 基础底布 黑色PP+绒+特种网格三层加强 ...
- **问题说明**: 招标文件对人造草坪和减震垫提出了非常具体的技术参数组合（如“双开槽形直曲混织草”、“17000Dtex”、“黑色PP+绒+特种网格三层加强”等），并要求提供SGS检测报告。这些参数组合高度特定，可能仅对应少数品牌或特定生产工艺，排除了其他符合国家标准GB 36246-2018但采用不同工艺（如单丝、草丝形状不同、底布结构不同）的合格产品。这种参数设置具有明显的指向性，涉嫌以不合理条件限制或排斥潜在供应商。
- **来源动作ID**: GEN-A05
- **关联审查点**: [[wiki/25-风险审查点/技术参数或服务要求指向特定供应商]]；[[wiki/30-风险库/技术参数指向特定品牌或型号]]
- **审查依据**: 《政府采购法实施条例》第二十条；《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十五条
- **修改建议**: 删除具有指向性的具体参数组合（如“双开槽形直曲混织草”、“17000Dtex”等），改为引用国家标准GB 36246-2018中的关键性能指标（如拉伸强度、耐磨性、有害物质限量等），并允许投标人提供符合国标的等效产品证明。
- **是否需要人工复核**: 否

### 4. 样品要求直接指定品牌，违反公平竞争原则

- **结论类型**: 指向性参数/排斥供应商
- **风险等级**: 高
- **原文位置**: 第五章 招标内容及要求 第1277行；第1276行
- **原文内容**: 
  - 1277: ⑤品牌:Aokang
  - 1276: 样品要求... ⑤品牌:Aokang
- **问题说明**: 招标文件在样品要求中直接指定品牌为“Aokang”（康乐佳）。在政府采购中，除单一来源采购或经批准的特殊情况外，直接指定品牌属于典型的排斥潜在供应商行为，违反了公平竞争原则。除非采购人能证明该品牌具有不可替代的唯一性，否则该条款违规。
- **来源动作ID**: GEN-A05
- **关联审查点**: [[wiki/25-风险审查点/技术参数或服务要求指向特定供应商]]；[[wiki/30-风险库/技术参数指向特定品牌或型号]]
- **审查依据**: 《政府采购法实施条例》第二十条；《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十五条
- **修改建议**: 删除指定品牌“Aokang”的要求。改为要求样品符合招标文件规定的技术参数和国家标准，允许投标人提供符合要求的任意品牌样品。
- **是否需要人工复核**: 否

### 5. 合同模板错配，关键条款空白

- **结论类型**: 合同条款缺陷
- **风险等级**: 中高
- **原文位置**: 第六章 政府采购合同 第1343-1687行；第1452行；第1431-1434行
- **原文内容**: 
  - 1343-1687: 第六章 政府采购合同参考文本（标题为“政府采购货物买卖合同（试行）”）
  - 1452: （6）履约验收标准：_____________________________
  - 1431-1434: 收取履约保证金形式：____________________ 收取履约保证金金额：____________________
- **问题说明**: 
  1.  **模板错配**: 本项目为运动场地维护（工程/服务混合），但合同模板为“政府采购货物买卖合同（试行）”，未体现工程/服务类合同的关键要素（如工期、施工进度、隐蔽工程验收等），存在模板错配风险。
  2.  **关键条款空白**: 合同专用条款中“履约验收标准”、“履约保证金形式”、“履约保证金金额”等关键字段为空白。虽然第五章有相关规定，但合同作为最终履约依据，其关键条款空白可能导致履约争议，且与第五章内容未形成有效衔接（未作为附件或专用条款填入）。
- **来源动作ID**: GEN-A07
- **关联审查点**: [[wiki/25-风险审查点/合同关键空白或模板错配]]；[[wiki/25-风险审查点/合同条款与采购需求不一致]]
- **审查依据**: 《政府采购需求管理办法》；《政府采购货物和服务招标投标管理办法》（财政部令第87号）第二十条
- **修改建议**: 替换为适用于工程/服务类项目的合同模板，或修改现有模板以匹配本项目属性。在合同专用条款中填入具体的履约验收标准、履约保证金形式和金额，确保合同条款与招标文件第五章要求完全一致。
- **是否需要人工复核**: 否

### 6. 证明材料要求不一致且过度，原件备查后果过重

- **结论类型**: 证明材料要求不当
- **风险等级**: 中
- **原文位置**: 第五章 招标内容及要求 第0950行、第1097行、第1221行、第1243行、第1339行
- **原文内容**: 
  - 0950: ...（提供有资质的检测机构出具的具有CMA**且**CNAS标志检测报告复印件为考评依据）
  - 1097: ...（提供有资质的检测机构出具的具有CMA**或**CNAS标志检测报告复印件为考评依据）
  - 1221: ...同时能够在全国认证认可信息公共服务平台网站可查询到，**或提供截图**
  - 1243: ...（提供SGS检测报告复印件为考评依据）
  - 1339: ...中标候选人无法提供原件复核的，**按提供虚假材料谋取中标或成交的情形处理**...
- **问题说明**: 
  1.  **要求不一致**: 不同材料对检测报告的要求不一致（0950行要求CMA**且**CNAS，1097行要求CMA**或**CNAS），导致评审标准不统一，易引发争议。
  2.  **截图证明效力**: 1221行允许提供“截图”作为检测报告查询证明，截图易篡改，真实性难以保证，不符合政府采购证明材料严谨性要求。
  3.  **SGS报告效力**: 1243行要求提供SGS（商业检测机构）报告，其作为符合国标GB 36246-2018的强制性依据效力存疑，除非明确认可其等效性。
  4.  **后果过重**: 1339行规定“无法提供原件复核”即按“提供虚假材料”处理，处罚过重。“无法提供原件”可能因遗失、管理不善等非主观恶意原因导致，直接认定为虚假材料缺乏合理性，建议改为“视为放弃中标资格”或“投标无效”。
- **来源动作ID**: GEN-A10
- **关联审查点**: [[wiki/25-风险审查点/证明材料要求不清或过度]]；[[wiki/25-风险审查点/原件备查或线下核验要求不当]]
- **审查依据**: 《政府采购质疑和投诉办法》（财政部令第94号）；《政府采购货物和服务招标投标管理办法》（财政部令第87号）
- **修改建议**: 
  1.  统一检测报告要求，建议仅要求具有CMA标志的检测报告（法定强制），或明确“CMA或CNAS”均可。
  2.  删除“或提供截图”的表述，改为要求提供官方查询结果打印件或原件/复印件。
  3.  明确SGS报告是否被认可为符合国标的等效证明，或改为要求具有CMA/CNAS资质的国内检测机构报告。
  4.  修改1339行后果，将“按提供虚假材料...处理”改为“视为放弃中标资格”或“投标无效”，并允许供应商说明原因。
- **是否需要人工复核**: 否

### 7. 专门面向中小企业采购与价格扣除规则冲突

- **结论类型**: 政策适用错误/规则冗余
- **风险等级**: 中
- **原文位置**: 第四章 资格审查与评标 第0651行、第0656-0660行
- **原文内容**: 
  - 0651: 采购包1：最低评标价法
  - 0656-0660: 中小企业价格扣除规则...（具体扣除比例和计算方式）
- **问题说明**: 本项目明确“专门面向中小企业采购”（0036行），意味着所有合格投标人均为中小企业。在此情况下，再设置“中小企业价格扣除”规则（0656-0660行）属于规则冗余，且可能导致评审混乱（所有投标人均享受扣除，扣除后排序不变，但增加了计算步骤和潜在争议）。根据《政府采购促进中小企业发展管理办法》，专门面向中小企业采购的项目，不再进行价格扣除。
- **来源动作ID**: GEN-A12
- **关联审查点**: [[wiki/25-风险审查点/中小企业政策适用错误]]；[[wiki/25-风险审查点/价格分和报价规则异常]]
- **审查依据**: 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）；《关于进一步加大政府采购支持中小企业力度的通知》（财库〔2022〕19号）
- **修改建议**: 删除专门面向中小企业采购项目中的价格扣除条款，或明确说明“因本项目专门面向中小企业采购，故不适用价格扣除政策”。
- **是否需要人工复核**: 否

### 8. 合同履约验收标准空白，引用标准过时

- **结论类型**: 验收标准缺失/不明确
- **风险等级**: 中高
- **原文位置**: 第六章 政府采购合同 第1452行；第五章 招标内容及要求 第0945行
- **原文内容**: 
  - 1452: （6）履约验收标准：_____________________________
  - 0945: ...场地的设计及验收符合以下已实施的国家标准：人工材料体育场地使用要求及检验方法（GB/T20033-2006）...
- **问题说明**: 
  1.  **合同空白**: 合同模板中“履约验收标准”栏为空白，未填写具体验收指标，导致合同履约依据不清。
  2.  **标准过时**: 招标文件引用的GB/T20033-2006标准已废止，现行有效标准为GB 36246-2018《中小学合成材料面层运动场地》。虽然0945行提到“以最新标准为准”，但合同空白仍构成风险。
- **来源动作ID**: GEN-A06
- **关联审查点**: [[wiki/25-风险审查点/履约验收标准缺失或不可操作]]
- **审查依据**: 《政府采购需求管理办法》；《政府采购货物和服务招标投标管理办法》（财政部令第87号）
- **修改建议**: 在合同模板的“履约验收标准”栏中明确填写具体的验收标准，优先引用现行有效的GB 36246-2018标准，并明确验收程序、方法和不合格处理机制。
- **是否需要人工复核**: 否
