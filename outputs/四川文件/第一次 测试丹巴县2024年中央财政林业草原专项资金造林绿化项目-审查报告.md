# 1. 审查摘要

本次审查针对《丹巴县2024年中央财政林业草原专项资金造林绿化项目》采购需求文件进行合规性审查。文件类型为政府采购项目采购需求，采购方式为公开招标，评审方法为综合评分法，预算金额2,899,600.00元，最高限价2,680,443.18元，专门面向中小企业采购（预留比例100%）。

审查发现主要风险集中在以下三个方面：
1.  **评分办法存在非相关资质加分**：评分办法中设置了“软件企业认定证书”和“ITSS信息技术服务标准符合性证书”加分项，与造林绿化项目需求无关，涉嫌以不合理条件限制或排斥供应商。
2.  **中小企业政策适用存在歧义**：项目品目为农林牧渔业，但部分标的物（肥料、农药等）行业属性判定为工业，且未明确中小企业声明函的填报规则，存在政策适用争议风险。
3.  **履约验收标准核心指标缺失**：虽然采购需求中提到了造林成活率指标，但在合同管理及履约验收方案章节中未将其作为明确的量化验收标准列出，仅笼统引用行业标准，导致验收标准可操作性不足。

# 4. 风险点清单

### 风险 1：采购标的所属行业属性判定错误导致中小企业政策适用争议
- **结论类型**: 高风险
- **风险等级**: 高
- **原文位置**: 0155行、0245行、0328行、0451行、0485行
- **原文内容**: 
  - 0155行：标的物所属行业 农、林、牧、渔业
  - 0245行：标的物所属行业 工业
  - 0328行：标的物所属行业 工业
  - 0451行：标的物所属行业 工业
  - 0485行：标的物所属行业 工业
- **问题说明**: 本项目为专门面向中小企业采购，品目统一为A07039900（其他农林牧渔业产品）。但文件中有机肥、复合肥、标识牌、多菌灵、敌百虫等标的物的所属行业被判定为“工业”。根据《政府采购促进中小企业发展管理办法》，中小企业声明函需按标的所属行业填写。苗木属于农林牧渔业，而肥料、农药、标识牌属于工业。文件未明确说明是否允许供应商针对工业品部分提供工业类中小企业声明函，或是否统一要求农林类声明函。这种行业属性混用且未明确声明函填报规则的情况，可能导致供应商对中小企业声明函填报产生困惑，进而影响投标有效性。
- **关联动作ID**: GEN-A01
- **关联审查点**: [[wiki/25-风险审查点/采购标的所属行业错误或缺失]]
- **审查依据**: 《政府采购促进中小企业发展管理办法》第四条、第八条；《政府采购品目分类目录》
- **修改建议**: 明确本项目所有标的物的中小企业行业属性判定标准。若允许混合，需在文件中明确“供应商可分别提供对应标的物的中小企业声明函”；若统一属性，需修正所有工业品标的物的行业属性为农林牧渔业（若符合实际）或明确排除工业品部分的中小企业预留。
- **是否需要人工复核**: 是

- **风险等级**: 高

- **问题说明**: ITSS证书通常用于信息化系统集成或运维服务，与林业造林绿化项目无关。评分办法中设置该证书作为商务评分项，与本项目采购需求（造林、管护）及履约质量无直接关联。此类非相关资质加分属于评审因素与采购需求脱节，构成以不合理的条件对供应商实行差别待遇或歧视待遇。
- **关联动作ID**: GEN-A04

- **审查依据**: 《政府采购货物和服务招标投标管理办法》（财政部令第87号）第五十五条；《政府采购法实施条例》第二十条
- **修改建议**: 删除“ITSS信息技术服务标准符合性证书”加分项，或替换为与林业服务、园林绿化养护直接相关的资质或业绩证明。
- **是否需要人工复核**: 是

### 风险 4：核心量化验收指标（成活率、保存率）未在验收方案中明确
- **结论类型**: 中高风险
- **风险等级**: 中高
- **原文位置**: 0016行、1099-1105行、1157-1172行
- **原文内容**: 
  - 0016行：人工造林当年造林成活率达到85%以上，株数保存率达80%以上
  - 1099-1105行：验收标准：符合国家、行业标准、四川省地方标准规定的验收标准。严格按照《财政部关于进一步加强政府采购需求和履约验收管理的指导意见》（财库〔2016〕205号）及《政府采购需求管理办法》（财库〔2021〕22号）的要求组织验收。验收应以采购合同、采购文件及其补充文件、国家或行业相关标准为验收的主要依据。
  - 1157-1172行：技术履约验收内容：逐条验收；如出现未在采购文件中明确规定的，以行业相关标准为准。...履约验收标准：按采购文件要求、中标（成交）人的投标（响应）文件及承诺、签订的合同、国家及行业相关规范标准进行。
- **问题说明**: 虽然0016行提到了“人工造林当年造林成活率达到85%以上，株数保存率达80%以上”，但在合同管理安排和履约验收方案章节中，未将这一核心量化指标明确写入验收标准的具体条款中，而是笼统地引用“国家、行业标准”和“采购文件要求”。对于林业项目，成活率和保存率是核心验收指标，若未在验收方案中明确具体的验收方法、时间节点（如当年、次年）、补植要求及对应的付款/违约责任挂钩机制，将导致验收标准不可操作，易引发履约争议。
- **关联动作ID**: GEN-A06
- **关联审查点**: [[wiki/25-风险审查点/履约验收标准缺失或不可操作]]
- **审查依据**: 《政府采购需求管理办法》第二十四条；《政府采购货物和服务招标投标管理办法》（财政部令第87号）第四十五条
- **修改建议**: 在履约验收方案中明确列出核心量化指标（成活率85%、保存率80%），并规定具体的验收时间、验收方法、不合格处理措施（如补植、扣款）及复验机制。
